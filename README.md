# NIST RMF Foundations Portfolio

This repository contains NIST Risk Management Framework (RMF) artifacts
created while completing the Coursera NIST DoD RMF course.

## Role Simulated
Security Control Assessor (SCA)

## Scenario
Fictional healthcare system assessed to demonstrate RMF lifecycle
outputs from a Security Control Assessor perspective.

## RMF Steps Covered
1. System Categorization
2. Control Selection
3. Control Implementation
4. Control Assessment
5. Authorization
6. Continuous Monitoring
